Definition (Connected Graph). A graph is connected if every pair of vertices is joined by a path.

Definition (Bridge). An edge is a bridge if deleting it increases the number of connected components.

Definition (2-Edge-Connected Graph). A graph is 2-edge-connected if it is connected and has no bridge.

Lemma (Bridge Characterization). In a connected graph, an edge is a bridge exactly when it is contained in no cycle.

Theorem (Cycle Characterization of 2-Edge-Connectivity). A connected graph is 2-edge-connected exactly when every edge lies on a cycle.
