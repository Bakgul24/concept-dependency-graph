Definition (Continuity). A function f is continuous at c if for every epsilon > 0 there exists delta > 0 such that |x-c| < delta implies |f(x)-f(c)| < epsilon.

Definition (Closed Interval). A closed interval [a,b] contains every real number x with a <= x <= b.

Lemma (Sign Change Lemma). If f is continuous on [a,b] and f(a) and f(b) have opposite signs, then f has a zero in [a,b].

Theorem (Intermediate Value Theorem). Theorem use sign change lemma. If f is continuous on [a,b] and y lies between f(a) and f(b), then there exists c in [a,b] such that f(c)=y.
